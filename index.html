This block diagram shows a scalable path from a static personal site to a richer platform with CMS, media, and login.

Recommended stack

Build in layers

  • Presentation layer: reusable UI components and page templates
  • Content layer: markdown or headless CMS for posts and page data
  • Media layer: image and video hosting with optimized delivery
  • Auth layer: backend-controlled login, sessions, and protected user actions. Hello Somu!
